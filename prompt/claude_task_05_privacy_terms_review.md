# Claude Task 05: 개인정보 및 약관 초안 보강

## 목표

Crewith 1차 MVP 운영 전에 필요한 개인정보 처리방침과 이용약관 초안을 서비스 범위에 맞게 보강한다.

## 작업 범위

수정할 파일:

- `docs/PRIVACY_POLICY_DRAFT.md`
- `docs/TERMS_DRAFT.md`

필요하면 새 파일을 추가할 수 있다.

- `docs/LEGAL_REVIEW_TODO.md`

## 세부 작업

### 1. 개인정보 처리방침 초안 보강

다음 항목을 포함한다.

- 수집하는 개인정보
- 필수/선택 수집 항목
- 휴대폰 인증 정보
- 회원 프로필 사진
- 모임 활동 정보
- 회비 납부 상태
- 일정 참석/출석 이력
- 공지 확인/댓글/좋아요 이력
- FCM 알림 토큰
- 개인정보 보관 기간
- 탈퇴 후 30일 삭제 정책
- 회비/참석 이력 보존 정책
- 제3자 제공 여부
- 위탁 처리 가능성
- 이용자 권리
- 문의 채널

### 2. 이용약관 초안 보강

다음 항목을 포함한다.

- 서비스 목적
- 회원 계정
- 모임 생성/운영
- 운영진 권한
- 회원 의무
- 게시글/댓글 책임
- 회비 정보의 성격
- 실제 결제 미제공 범위
- 알림 수신
- 서비스 제한/중단
- 탈퇴/강퇴/삭제
- 면책
- 약관 변경

### 3. 법무 검토 TODO

`docs/LEGAL_REVIEW_TODO.md`를 생성한다면 아래를 포함한다.

- 실제 서비스 출시 전 법무 검토 필요 항목
- 개인정보보호법 관점 확인 필요 항목
- 전자상거래/결제 도입 시 추가 필요 항목
- 미성년자 이용 가능성 검토
- 스포츠 모임 사고/책임 관련 면책 검토
