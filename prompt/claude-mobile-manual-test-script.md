# Claude 독립 작업: 회원 앱 수동 테스트 스크립트 작성

## 목표
Flutter 회원 앱을 실제 기기/에뮬레이터에서 확인할 때 사용할 수 있는 수동 테스트 스크립트를 작성한다.

## 작업 범위
- `apps/mobile-app/README.md`
- `docs/MEMBER_APP_FLOW.md`
- 필요하면 `docs/QA_CHECKLIST.md`

## 지시
1. 회원 앱 실행 전 준비사항을 정리한다.
   - API 서버 실행
   - 앱 실행 명령
   - 기본 테스트 휴대폰 번호와 OTP
   - 기본 테스트 회원과 모임
2. 화면별 수동 테스트 절차를 작성한다.
   - 인증
   - 홈
   - 모임 전환
   - 일정 참석/불참 변경
   - 공지 확인 처리
   - 회비 확인
   - 프로필 저장
   - 공개 모임 가입 신청
   - 초대 코드 가입
3. 기대 결과와 실패 시 확인할 로그/파일을 함께 적는다.
4. 코드 변경은 하지 말고 문서만 수정한다.

## 완료 기준
- 처음 보는 개발자가 README만 보고 회원 앱을 실행하고 주요 기능을 확인할 수 있다.
- `docs/MEMBER_APP_FLOW.md`와 내용이 충돌하지 않는다.
- 완료 후 이 prompt 파일은 삭제한다.
