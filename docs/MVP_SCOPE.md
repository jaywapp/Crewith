# Crewith MVP 범위표

## 1. MVP 목표

Crewith MVP는 10~30명 규모의 스포츠 동호회가 구성원, 회비, 일정, 공지사항을 하나의 서비스에서 관리할 수 있는지 검증한다.

핵심 검증 목표는 운영진의 반복 관리 부담을 줄이고, 일반 회원이 본인의 일정, 회비, 공지 정보를 한 곳에서 확인할 수 있게 만드는 것이다.

## 3. MVP 포함하되 단순 구현

| 영역 | 기능 | 단순화 기준 |
|---|---|---|
| 결제 | 구독 상태 | 실제 결제 없이 요금제 안내와 상태 표시 |
| 회비 | 납부 기록 | 실제 결제 없이 운영진 수동 기록 |
| 개인정보 삭제 | 탈퇴/강퇴/휴면 처리 | 개인정보는 30일 후 삭제, 회비/참석 이력 보존 |
| 파일 | 프로필 사진 | 회원 프로필 사진만 지원 |
| 권한 | 세부 권한 | 모임장/운영진/일반회원 3단계로 제한 |
| 관리자 | 웹 관리자 | 회원/회비/일정/공지/출석/통계 중심 |
| 앱 | 운영진 기능 | 모바일 앱에서도 주요 운영 기능 제공 |

## 4. MVP 제외 범위

| 영역 | 제외 기능 | 사유 |
|---|---|---|
| 결제 | 실제 구독 결제 | MVP 이후 연동 |
| 회비 | 서비스 내 회비 결제 | 납부 기록/미납 관리에 집중 |
| 회비 | 일정 참석자 대상 자동 부과 | 1차 MVP 복잡도 축소 |
| 일정 | 회원 직접 체크인 | 운영진 체크로 시작 |
| 모임 | 공개 모임 검색/탐색 | 초기에는 가입 신청/초대 링크 중심 |
| 회원 | 회원별 포지션/역할 관리 | 종목은 모임 단위로 관리 |
| 권한 | 복잡한 커스텀 권한 체계 | 3단계 역할로 시작 |
| 커뮤니티 | 실시간 채팅 | 카카오톡 대체까지 확장하지 않음 |
| 분석 | 고급 BI | 핵심 운영 지표만 제공 |
| 글로벌 | 다국어/글로벌 운영 | 국내 스포츠 동호회 우선 |
| AI | 추천/자동화 AI | MVP 검증 범위 외 |
| 파트너 | 외부 파트너 포털 | MVP 범위 외 |

## 5. 추후 확장 후보

- PASS, NICE, KMC 등 국내 본인인증 연동
- iOS 정식 검증 및 출시
- 실제 구독 결제
- 회원 직접 체크인
- 일정 참석자 대상 회비 자동 부과
- 공개 모임 검색/탐색
- 회원별 포지션/역할 관리
- 회계 담당자, 일정 담당자 등 세부 운영진 권한
- 공지 첨부파일, 일정 사진, 회비 증빙 업로드
- 고급 통계와 리포트

## 6. MVP 성공 기준 초안

- 운영진이 Excel 없이 회원 명부를 등록하고 관리할 수 있다.
- 운영진이 월회비와 일회성 비용의 납부 상태를 추적할 수 있다.
- 운영진이 일정별 참석 의사와 실제 출석부를 관리할 수 있다.
- 회원이 앱에서 본인의 회비, 일정, 공지를 확인할 수 있다.
- 공지 확인률, 출석률, 회비 수납률이 대시보드에서 확인된다.
- 10~30명 규모 스포츠 동호회 1개 이상이 실제 운영에 사용할 수 있다.
