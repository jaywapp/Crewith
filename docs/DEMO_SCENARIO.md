# Crewith MVP 데모 시나리오

## 데모 목적

Crewith 1차 MVP를 처음 보는 사람이 관리자 웹과 회원 앱의 핵심 가치를 15분 안에 이해하고 체험할 수 있도록 한다.

**핵심 메시지**: Excel 장부, 카카오톡 공지, 카카오 캘린더로 나눠 관리하던 동호회 운영을 하나의 서비스에서 처리한다.

---

## 데모 대상 사용자

- 스포츠 동호회 운영진 (모임장, 총무, 회무 담당)
- Crewith 투자자/파트너 검토자
- 신규 개발팀 온보딩

---

## 사전 준비

### 환경

| 항목 | 설정값 |
|---|---|
| 관리자 웹 | http://localhost:3000 (또는 배포 URL) |
| API 서버 | http://localhost:4000 |
| 데모 모임명 | 토요 풋살 |
| 데모 운영진 번호 | 01011110001 |
| 데모 회원 번호 | 01011110002 |
| OTP (개발) | 123456 |

### 데모 데이터 초기 상태 (권장)

- 회원 8명 등록 (활성 6명, 휴면 2명)
- 이번 달 월회비 항목 1개 (납부 4명, 미납 2명)
- 다음 주 일정 1개 (참석 3명, 불참 1명, 미응답 2명)
- 공지 1개 (확인 5명, 미확인 1명)
- 가입 신청 대기 1건

---

## 데모 흐름

### Step 1. 관리자 웹 접속 (1분)

1. 브라우저에서 관리자 웹 URL 접속
2. 로그인 화면 확인
3. 운영진 번호 `01011110001` 입력
4. OTP `123456` 입력
5. 대시보드 진입

> **포인트**: "기존에는 Excel, 카카오톡, 수기 장부로 관리했던 내용이 이 화면 하나에 모입니다."

---

### Step 2. 대시보드 확인 (2분)

1. 상단 지표 카드 설명
   - 전체 회원 8명, 활성 회원 6명
   - 미납자 2명
   - 공지 확인률 83%
   - 실제 출석률 (지난 일정 기준)
2. 작업 큐 패널 설명
   - 가입 신청 대기 1건
   - 회비 미납자 2명
   - 다음 일정 날짜

> **포인트**: "운영진이 출근하면 이 화면 하나로 오늘 처리해야 할 일을 바로 파악할 수 있습니다."

---

### Step 3. 회원 추가 (2분)

1. 사이드바 > 구성원 진입
2. 회원 추가 버튼 클릭
3. 이름: `김시연`, 번호: `01099998888`, 역할: `일반 회원` 입력
4. 저장 후 목록 상단에 추가 확인
5. 역할을 운영진으로 변경해보기

> **포인트**: "초대 링크를 보내거나 직접 추가할 수 있습니다. 역할도 즉시 변경됩니다."

---

### Step 4. 회비 생성 및 납부 상태 변경 (3분)

1. 사이드바 > 회비 진입
2. 이번 달 월회비 항목 확인 (미납 2명 표시)
3. 미납자 중 1명의 납부 상태 토글 (미납 → 납부)
4. 납부일이 오늘 시각으로 자동 기록되는 것 확인
5. 일회성 비용 생성
   - 항목명: `대관비`, 금액: `5,000원`, 납부일: 이번 주 일요일
   - 대상: 전체 회원 선택 후 저장

> **포인트**: "수기 장부에 체크하던 것을 클릭 한 번으로 처리합니다. 납부일도 자동으로 기록됩니다."

---

### Step 5. 일정 생성 및 참석 상태 확인 (2분)

1. 사이드바 > 일정 진입
2. 다음 주 일정 클릭
3. 참석 의사 탭 확인 (참석 3 / 불참 1 / 미응답 2)
4. 일정 생성 버튼으로 새 일정 빠르게 생성
   - 제목: `번개 경기`, 일시: 이번 주 토요일 오전 10시
5. 저장 후 목록 확인

> **포인트**: "참석 의사를 사전에 받아서 인원 파악이 가능합니다. 카카오톡에서 일일이 답장 세는 것과 다릅니다."

---

### Step 6. 출석 체크 (1분)

1. 지난 일정 클릭 > 출석부 탭
2. 회원별 출석/지각/결석 선택
3. 동반 인원 수 입력
4. 저장 후 출석률 확인

> **포인트**: "실제 출석률을 기록해두면 나중에 운영 통계에 활용할 수 있습니다."

---

### Step 7. 공지 작성 및 확인률 확인 (2분)

1. 사이드바 > 공지 진입
2. 기존 공지 클릭 > 확인 현황 탭
3. 확인 5명 / 미확인 1명 표시 확인
4. 미확인자에게 리마인더 발송
5. 공지 작성 버튼으로 새 공지 작성
   - 제목: `이번 주 경기 안내`, 내용 입력 후 저장

> **포인트**: "공지를 읽었는지 안 읽었는지 확인할 수 있습니다. 읽지 않은 회원에게만 다시 알림을 보낼 수 있습니다."

---

### Step 8. 가입 신청 승인 (1분)

1. 사이드바 > 가입/초대 진입
2. 가입 신청 대기 1건 확인
3. 신청자 정보 (이름, 연락처, 가입 인사) 확인
4. 승인 클릭
5. 회원 목록에서 신규 회원 추가 확인

> **포인트**: "공개 모임의 경우 신청을 검토하고 승인할 수 있습니다."

---

### Step 9. 초대 링크 생성 (30초)

1. 가입/초대 > 초대 링크 탭
2. 링크 생성 (30일 유효)
3. 링크 복사 후 공유 시뮬레이션
4. 비활성화 버튼 설명

> **포인트**: "비공개 모임은 이 링크를 공유해서 가입시킵니다. 만료일 설정도 가능합니다."

---

### Step 10. 회원 앱 로그인 (1분)

1. 회원 앱 실행 (에뮬레이터 또는 실기기)
2. 번호 `01011110002` 입력
3. OTP `123456` 입력
4. 홈 화면 진입

> **포인트**: "회원 입장에서는 앱으로 모든 정보를 확인할 수 있습니다."

---

### Step 11. 회원 앱에서 일정/공지/회비 확인 (2분)

1. 홈 탭에서 다음 일정, 미납 회비, 미확인 공지 확인
2. 일정 탭 > 번개 경기 > 참석 응답
3. 공지 탭 > 공지 탭 > 확인 처리 (자동)
4. 회비 탭 > 본인 납부 상태 확인

> **포인트**: "회원은 본인에게 필요한 정보만 간단하게 확인하고 응답할 수 있습니다."

---

### Step 12. 모임 전환 UI 확인 (30초)

1. 홈 화면 상단 모임명 탭
2. 모임 목록 드롭다운 확인
3. 여러 모임에 가입된 경우 전환 가능함 설명

> **포인트**: "한 사람이 여러 동호회에 가입해도 각각 별도로 관리됩니다."
