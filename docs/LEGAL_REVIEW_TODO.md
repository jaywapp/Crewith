## 1. 개인정보보호법 관점 확인 필요 항목

### 1.1 개인정보 처리방침 공개 의무

| 항목 | 내용 |
|---|---|
| 근거 | 개인정보보호법 제30조 |
| 필요 조치 | 앱/웹 내 접근 가능한 위치에 개인정보 처리방침 게시 |
| MVP 영향도 | 높음 — 출시 전 필수 |

처리방침에 반드시 포함해야 할 사항:
- 수집하는 개인정보 항목 (필수/선택 구분)
- 수집 및 이용 목적
- 보유 및 이용 기간
- 제3자 제공 여부 및 대상
- 처리 위탁 현황
- 정보주체 권리 및 행사 방법
- 개인정보 보호책임자 성명/연락처

### 1.2 동의 수집 방식

| 항목 | 내용 |
|---|---|
| 근거 | 개인정보보호법 제15조, 제22조 |
| 필요 조치 | 회원가입(인증) 시 처리방침 동의 UI 구현 |
| MVP 영향도 | 높음 — 로그인 플로우에 포함 필요 |

확인 필요 사항:
- 필수/선택 동의 항목 분리 여부
- 선택 동의 거부 시에도 서비스 이용 가능 여부
- 동의 이력 저장 및 증빙 방법

### 1.3 개인정보 처리 위탁 고지

| 항목 | 내용 |
|---|---|
| 근거 | 개인정보보호법 제26조 |
| 필요 조치 | Firebase(Google), 네이버 지도 API 위탁 현황 처리방침에 명시 |
| MVP 영향도 | 중간 — 처리방침 보강으로 해결 가능 |

확인 필요 수탁자:
- Google LLC (Firebase Auth, Firestore, FCM, Storage)
- 네이버 (지도 API)
- Railway / Vercel (서버 호스팅)

### 1.4 개인정보 보호책임자 지정

| 항목 | 내용 |
|---|---|
| 근거 | 개인정보보호법 제31조 |
| 필요 조치 | 서비스 운영 조직 내 개인정보 보호책임자 지정 및 처리방침에 연락처 공개 |
| MVP 영향도 | 높음 — 출시 전 필수 |

### 1.5 만 14세 미만 이용자 처리

## 3. 스포츠 모임 서비스 특수 사항

### 3.1 활동 중 사고/책임 면책

| 항목 | 내용 |
|---|---|
| 배경 | Crewith는 일정, 출석, 회비를 기록하는 서비스로, 실제 스포츠 활동 사고에 대한 책임 귀속이 문제될 수 있음 |
| 확인 필요 | 서비스가 모임 활동 자체를 주선하거나 알선하는 역할을 하지 않음을 이용약관에 명확히 기재 |
| MVP 영향도 | 중간 — 이용약관 면책 조항 보강 필요 |

권고 문구 방향:
- Crewith는 모임 운영 도구를 제공할 뿐, 모임 활동 중 발생하는 사고나 분쟁에 관여하지 않는다.
- 모임 운영 및 활동에 관한 책임은 모임장 및 운영진에게 있다.

### 3.2 회비 정보의 성격

| 항목 | 내용 |
|---|---|
| 배경 | Crewith는 회비 납부 상태를 기록하지만 실제 결제를 처리하지 않음 |
| 확인 필요 | 회비 정보가 금융 거래 기록으로 해석될 여지가 있는지, 저장/표시 방식이 문제 없는지 |
| MVP 영향도 | 낮음 — 단순 기록 도구임을 명확히 하면 됨 |
